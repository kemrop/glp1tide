## 🚀 Getting Started

### 📦 Dependencies

- next 16.1+
- node v22.10+
- npm v10.2+
- tailwind v4.0+

### 👉 Install Dependencies

```bash
npm install
```

### 👉 Development Command

```bash
npm run dev
```

### 👉 Build Command

```bash
npm run build
```

### 👉 Build and Run With Docker

```bash
docker build -t glp1tide .
docker run -p 3000:3000 glp1tide
```
